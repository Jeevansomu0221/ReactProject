# ReactProject

PopX mobile app interface built with React and Vite. A pixel-perfect implementation of the onboarding, authentication, and account settings screens.

## Features

- Welcome screen with navigation to Sign Up and Login
- Sign In form with floating label inputs
- Create Account form with validation fields and agency selection
- Account Settings profile page
- Mobile-first design centered on desktop viewport
- React Router for seamless page navigation

## Tech Stack

- React 19
- Vite
- React Router DOM

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
